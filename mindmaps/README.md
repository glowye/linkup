# Mind Maps Directory

This directory contains mind map images for the Communication Toolkit section.

## Required Files

1. `listening-mindmap.png` - How to Listen to Improve Communication Efficiency
2. `responding-mindmap.png` - How to Respond to Guide Others' Needs
3. `reading-people-mindmap.png` - How to Read People to Use Communication Strategies
4. `asking-help-mindmap.png` - How to Ask for Help to Gain More Resources
5. `refusing-mindmap.png` - How to Refuse to Manage Relationship Boundaries
6. `persuading-mindmap.png` - How to Persuade Others to Support You

## Format

- Format: PNG (recommended) or JPG
- Recommended size: 1200x800px or larger
- Style: Similar to the listening mind map example provided

## Adding Mind Maps

1. Create or export your mind map as an image
2. Save it with the corresponding filename above
3. Place it in this `mindmaps/` directory
4. The download functionality will automatically work
